# NeoForge modpack

Minecraft 1.21.1 modpack using NeoForge 21.1.230 and Packwiz.

## Manual CurseForge downloads

Two mods prohibit automatic third-party downloads. Download the exact files from
CurseForge and place them in the installation's `mods/` directory, then rerun the
Packwiz installer. Packwiz will verify the files and continue.

### Client

- [Create Recycling Everything 1.0.1](https://www.curseforge.com/minecraft/mc-mods/create-recycling-everything-neo/files/6383172)
  - File: `create_recycle_everything-1.0.1.jar`
  - SHA-1: `405b43418fded03ef391a9dc9ee3e12a11145e0d`

### Dedicated server

Install the client file above and:

- [Create Colony Logistics 1.2.2](https://www.curseforge.com/minecraft/mc-mods/cclogistics/files/7904223)
  - File: `create_colony_logistics-1.2.2.jar`
  - SHA-1: `9f60e59d429e637a73095d4960c3f838edfa7b40`

These versions are intentionally retained. Updating either mod requires another
manual download and a Packwiz metadata refresh.
